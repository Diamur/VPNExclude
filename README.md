# VPNExclude

Утилита для Windows, которая помогает добавлять маршруты обхода VPN для отдельных доменов/IP при активном WireGuard full tunnel.

## Идея
По умолчанию весь трафик идёт через WireGuard.

Для отдельных доменов/ресурсов можно:
- определить их текущие IP
- добавить постоянные маршруты через локальный шлюз
- тем самым пустить их мимо VPN

## Основные сценарии
- school.mos.ru
- mos.ru
- gosuslugi.ru
- другие ресурсы, которые должны идти не через VPN

## Файлы
- `VPNExclude.ps1` — основной PowerShell-скрипт
- `domains.txt` — список доменов для обработки
- `logs/routes.log` — журнал добавленных маршрутов
