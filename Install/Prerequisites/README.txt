Place prerequisite installers in this folder before building setup:

- wireguard-installer-x64.exe
- windowsdesktop-runtime-8.0.x-win-x64.exe

Notes:
- For WinForms framework-dependent deployment, use Windows Desktop Runtime 8 x64.
- For self-contained publish, .NET runtime installer is optional.
- Do not assume silent switches for WireGuard unless confirmed from official source.
